HOW TO USE TM:

Launch the TM as a jar file or with launch arguments in IDE with arguments as follows:

<command to launch jarfile> <address:port> <address:port>

Where address:port is something like localhost:5000 etc etc.


HOW TO USE QB

Execute with the python interpreter with 2 arguments

python3 WhereeverDirectoryItIs/QB_solution.py <address> <port>

where address is something like localhost
and port is something like 50000
